# 法律相关问题清单（共20个不同领域）

| 序号 | 法律领域         | 相关问题                                                     |
| :--: | :--------------- | :----------------------------------------------------------- |
|  1   | **婚姻家庭**     | 一方患有重大疾病但未在婚前告知另一方，另一方可以怎么办？     |
|  2   | **财产继承**     | 胎儿是否享有继承权？                                         |
|  3   | **合同纠纷**     | 对保证方式没有约定，如何确定保证的方式？                     |
|  4   | **物权保护**     | 住宅建设用地使用权期限届满，如何处理？                       |
|  5   | **侵权责任**     | 因自愿实施紧急救助造成受助人损害的，救助人是否要承担责任？   |
|  6   | **劳动用工**     | 用人单位拖欠农民工工资可能面临什么后果？                     |
|  7   | **消费者权益**   | 对于试用买卖，试用期限届满买方未作表示，会产生什么效果？     |
|  8   | **未成年人保护** | 未成年人遭受性侵害，诉讼时效从何时开始计算？                 |
|  9   | **网络虚拟财产** | 法律是否保护游戏账号、虚拟货币等网络虚拟财产？               |
|  10  | **房屋租赁**     | 租赁期限最长不得超过多少年？超过部分是否有效？               |
|  11  | **物业服务**     | 业主能否以未接受物业服务为由拒绝支付物业费？                 |
|  12  | **肖像权**       | 为个人学习，可以使用他人已经公开的肖像吗？                   |
|  13  | **姓名权**       | 除随父姓或随母姓外，可以选取其他直系长辈血亲的姓氏吗？       |
|  14  | **收养关系**     | 无配偶者收养异性子女，年龄应相差多少岁？                     |
|  15  | **诉讼时效**     | 向法院请求保护民事权利的普通诉讼时效是多久？                 |
|  16  | **担保物权**     | 定金数额不得超过主合同标的额的多少？                         |
|  17  | **交通安全**     | 未经允许驾驶他人机动车发生交通事故，责任由谁承担？           |
|  18  | **教育机构责任** | 无民事行为能力人在幼儿园、学校等教育机构受到人身损害，责任如何认定？ |
|  19  | **高空抛物**     | 高空抛物坠物需要承担什么法律责任？                           |
|  20  | **正当防卫**     | 正当防卫超过必要限度，是否要承担民事责任？                   |
